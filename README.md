# README

This project has been built using NestJS v9, to create a REST API with passport jwt authentication, swagger OAS3 documentation and MongoDB.
The client directory contains the build output of the `contacts_manager_client` project in my GitLab profile.
A running demonstration is available at [Vercel](https://nest-react-contacts.vercel.app/).

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
